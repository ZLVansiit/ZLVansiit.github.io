# 常家岩

这是我老家，叫常家岩。

如今那里就剩下老张家两户了。姓常的已经绝户，房子塌完了，一片荒寂里只剩断墙和野草。以前他家门口有一棵很大的樱桃树，花开时像一团粉云，结果时孩子们围着转；现在也败了，枝枯叶疏，再没人打理。

老张是五保户，村里给他在镇上安排了房子，好歹有个落脚处。五一我回去，还看见他——人瘦，话不多，眼神里是那种见过太多起落之后的平静。我顺着旧路上去，他仍往老地方走，种菜、种地，好像那块地比镇上的砖墙更亲。

我走过去，喊他一声，给他散了支烟。他接过，点上，我们站在塌屋边上抽完，谁也没多说什么。风从山坳里过来，带着土腥气和青草味，和小时候一样。

我常想，「常家岩」这三个字，当初是怎么落到地图和嘴上的。是姓常的先人住过这面岩下，还是岩形像什么、故事里有什么讲究，老一辈若还在，兴许能掰扯几句；等他们不在了，再过一代，大概就没人说得清这个名字的来历了。

地名和人一样，也会慢慢淡掉。我写下来，算给自己留个底——万一哪天连我也忘了，至少还有一行字证明：这里曾经叫常家岩，我曾经回去过，还曾给老张散过一支烟。
