# 常家岩

![img.png](https://vansiit.cc/moments/img.jpg)

这是我老家，叫常家岩。

五一回去，去了很多趟。

说是常家岩，其实就剩老张家（我奶奶和我小奶奶）两户人了。姓常的早些年就绝了户，房子塌得不成样子，只剩些零星的瓦砾和半截土墙。常大爷以前家门口那棵很大的樱桃树也败了， 熟成一颗被鸟吃一颗，再加上树荫遮住了，五一回去没见着几颗樱桃。

![img.png](https://vansiit.cc/moments/img-1.jpg)

这是在我奶奶家摘的。

常大爷现在是五保户，村里给他在镇上安排了房子。可我那天上去的时候，远远就看见他在老屋地基边上，弓着腰种菜。地不大，拾掇得倒挺齐整。我走过去喊了一声，他抬头看我，愣了好一会儿才认出来。

我给他散了根烟，他接过去。笑着脸，说回来了啊，还是那么熟悉的笑脸。

就想起小时候路过这里，樱桃熟了的时候，红艳艳的一树，老远就能看见。一回想也有二十年了。

再过一代人，恐怕真的没人知道“常家岩”这个名字是怎么来的了。
