# 常家岩

![img.png](https://vansiit.cc/img/changjiayan/img1.jpg)

这是我老家，叫常家岩。

五一回去，还特意上常家岩走了一趟。

说是常家岩，其实就剩老张家两户人了。姓常的早些年就绝了户，房子塌得不成样子，只剩些零星的瓦砾和半截土墙。常大爷以前家门口那棵很大的樱桃树，也败了，枯枝丫在风里颤颤巍巍的，看着让人心里头发酸。

常大爷现在是五保户，村里给他在镇上安排了房子。可我那天上去的时候，远远就看见他在老屋地基边上，弓着腰种菜。地不大，拾掇得倒挺齐整。我走过去喊了一声，他抬头看我，愣了好一会儿才认出来。

我给他散了根烟，他接过去，手有点抖。也没说什么话，就那么蹲在地头抽着，眼睛望着那片塌了的房子。

忽然就想起小时候路过这里，樱桃熟了的时候，红艳艳的一树，老远就能看见。

再过一代人，恐怕真的没人知道“常家岩”这个名字是怎么来的了。
